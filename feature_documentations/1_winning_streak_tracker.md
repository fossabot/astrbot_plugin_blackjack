# 功能开发文档：连胜追踪器

## 功能概述
记录并展示玩家的连胜次数，达到一定连赢单独奖励

## 需求分析
- 记录每个玩家的连续胜利次数
- 设置连胜奖励阈值
- 展示当前连胜数和历史最高连胜数
- 在达成连胜里程碑时发送祝贺信息

## 技术实现

### 数据模型修改
```python
# 在 models.py 中添加
@dataclass
class PlayerStreak:
    user_id: str
    current_streak: int = 0
    max_streak: int = 0
    last_game_time: float = 0
```

### 主要逻辑实现
1. 游戏结束后判断胜负结果
2. 更新玩家连胜记录
3. 检查是否达成连胜里程碑
4. 发放相应奖励
5. 保存到持久化存储

### 奖励机制
- 5连胜：获得10金币
- 10连胜：获得50金币 + 特殊称号
- 20连胜：获得200金币 + 独特称号
- 50连胜：获得1000金币 + 稀有称号

## 用户界面
- 游戏信息中显示当前连胜数
- 个人战绩页面显示历史最高连胜
- 连胜里程碑达成时弹出祝贺信息

## 测试要点
- 连胜计数准确性
- 败北时清零机制
- 连胜奖励发放
- 跨会话数据持久化
