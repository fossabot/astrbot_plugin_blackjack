# 功能开发文档：多人合作模式

## 功能概述
几个玩家组成团队对抗AI庄家，共享收益

## 需求分析
- 支持多玩家协作游戏
- 团队共同对抗庄家
- 收益按贡献度分配
- 支持邀请和组队功能

## 游戏模式设计

### 基础规则
- 2-4人团队合作
- 所有玩家共同对抗AI庄家
- 共享筹码池
- 玩家轮流做出决策

### 团队决策机制
- 主动权轮流制：每位玩家轮流做决策
- 投票制：对关键决策进行投票
- 队长制：队长拥有最终决策权

### 贡献度计算
- 基础贡献：下注金额
- 技能贡献：使用技能辅助团队
- 策略贡献：提出有效建议
- 协助贡献：帮助队友决策

## 技术实现

### 新增数据模型
```python
@dataclass
class TeamGameSession:
    team_id: str
    players: list[Player]
    captain: str
    pot: float
    contribution: dict[str, float] = field(default_factory=dict)
    team_strategy: str = "轮流出牌"
```

### 游戏流程调整
1. 队长创建队伍
2. 邀请玩家加入（设定加入门槛）
3. 所有队员共同下注
4. 按照团队策略轮流决策
5. 计算团队总收益
6. 按贡献度分配收益

### 邀请机制
- 通过ID邀请好友
- 二维码快速加入
- 自动匹配功能

## 用户界面
- `/team create` 创建团队
- `/team invite <player_id>` 邀请玩家
- `/team join <team_id>` 加入团队
- 团队聊天窗口

## 特殊功能
- 团队聊天功能
- 策略讨论时间
- 决策投票系统
- 团队成就系统

## 测试要点
- 多人协作流程正常
- 收益分配算法准确
- 团队决策机制有效
